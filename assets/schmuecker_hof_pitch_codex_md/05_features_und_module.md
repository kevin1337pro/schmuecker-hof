# 05 – Features und Module

## Pflichtmodule für den Pitch-Prototyp

### 1. Header mit CTA

Funktionen:

- Logo
- Navigation
- Reservierungsbutton
- Mobile Menü

### 2. Hero Section

Funktionen:

- großes Bild oder Video
- Headline
- Subheadline
- CTA-Buttons
- saisonaler Hinweis

### 3. Aktuell auf dem Hof

Karten für aktuelle Themen:

- Erdbeeren
- Spargel
- Hofmarkt
- Selbstpflücken
- Verwöhnabende
- Advent

Im Prototyp statisch. Später CMS-editierbar.

### 4. Erlebnis-Kacheln

Vier Kacheln:

- Hofküche
- Hofmarkt
- Hofplantagen
- Feiern

### 5. Speisekarten-Modul

Darstellung als Tabs:

- Frühstück
- Mittagessen
- Nachmittagskarte
- Abendkarte

Zusätzlich PDF-Download.

### 6. Saisonkalender

Interaktive Monatsübersicht.

Beispiel:

```json
{
  "Mai": ["Erdbeeren", "Spargel", "Selbstpflücken"],
  "Juni": ["Erdbeeren", "Blaubeeren", "Kirschen"],
  "Dezember": ["Advent", "Weihnachtsbäume", "Geschenke"]
}
```

### 7. Event-Anfrageformular

Felder:

- Name
- E-Mail
- Telefon
- Art der Feier
- Wunschtermin
- Gästezahl
- Nachricht
- Datenschutz-Checkbox

### 8. Job-Kurzbewerbung

Felder:

- Name
- Telefon
- E-Mail
- Bereich
- Verfügbarkeit
- Nachricht
- optional Lebenslauf-Upload

### 9. Kontaktmodul

- Adresse
- Telefon
- E-Mail
- Öffnungszeiten
- Maps-Link
- Social Links

## Erweiterungsmodule für spätere Version

### Online-Reservierung

Anbindung an ein Reservierungstool oder individuelles Formular mit E-Mail-Versand.

### CMS für saisonale Inhalte

Redaktion kann aktuelle Angebote selbst pflegen.

### Instagram Feed

Darstellung aktueller Beiträge.

### Hofmarkt-Vorbestellung

Später möglich für:

- Spargel
- Erdbeeren
- Präsentkörbe
- Kuchen

### Newsletter

Saisonale Angebote und Events per E-Mail.

### Digitale Gutscheine

Gutscheinverkauf oder Gutscheinanfrage.
