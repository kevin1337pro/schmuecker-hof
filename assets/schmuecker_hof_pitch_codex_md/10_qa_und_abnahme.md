# 10 – QA und Abnahme

## Visuelle Qualität

Die Website ist abnahmebereit, wenn:

- sie hochwertig, freundlich und modern wirkt
- die Marke Schmücker Hof klar erkennbar bleibt
- Bilder groß und emotional eingesetzt werden
- Abstände und Typografie konsistent sind
- die Website nicht wie ein Standard-Template wirkt

## Inhaltliche Qualität

Prüfen:

- Sind alle Hauptbereiche enthalten?
- Sind CTA-Texte klar?
- Sind Öffnungszeiten sichtbar?
- Sind Kontaktinformationen leicht auffindbar?
- Werden Speisekarten nutzerfreundlich dargestellt?
- Ist der Eventbereich hochwertig genug?
- Ist die Jobs-Seite niedrigschwellig?

## Mobile Qualität

Prüfen auf Smartphone:

- Header bedienbar
- Menü gut lesbar
- CTAs gut erreichbar
- Formulare einfach ausfüllbar
- Karten nicht zu klein
- Footer übersichtlich
- keine horizontalen Scrollprobleme

## Conversion-Check

Wichtige Fragen:

- Kann ein Nutzer innerhalb von 10 Sekunden Öffnungszeiten finden?
- Kann ein Nutzer schnell eine Reservierung starten?
- Ist die Hochzeitsanfrage prominent genug?
- Ist der Kontakt immer erreichbar?
- Sind Saisonangebote auf der Startseite sichtbar?
- Ist die Jobbewerbung einfach genug?

## SEO-Check

Für jede Seite prüfen:

- eindeutiger Seitentitel
- Meta Description
- H1 vorhanden und nur einmal
- sinnvolle H2-Struktur
- sprechende URL
- lokale Keywords
- Alt-Texte für Bilder

## Technik-Check

Ausführen:

```bash
npm run lint
npm run build
```

Akzeptanz:

- Build läuft fehlerfrei
- keine kritischen Lintfehler
- keine kaputten Imports
- keine ungenutzten Hauptkomponenten

## Barrierefreiheit Basischeck

Prüfen:

- ausreichende Kontraste
- Buttons mit klaren Labels
- Formularfelder mit Labels
- Fokuszustände sichtbar
- Bilder mit Alt-Text
- Navigation per Tastatur möglich

## Pitch-Abnahme

Für den Pitch reicht ein klickbarer Prototyp mit:

- Startseite
- Hofküche
- Hofmarkt
- Hofplantagen
- Feiern & Hochzeiten
- Saison & Specials
- Jobs
- Kontakt

Der Prototyp muss nicht alle echten Daten enthalten, aber er muss das neue Erlebnis glaubwürdig zeigen.
