# 03 – Designsystem

## Designrichtung

Die Website soll hochwertig, freundlich, modern und natürlich wirken. Das bestehende Logo-Blau bleibt erhalten, wird aber mit warmen Naturtönen kombiniert.

Leitgefühl:

> Tradition im Kern, Moderne im Auftritt.

## Farbwelt

### Primärfarbe

**Schmücker Blau**

- Verwendung: Navigation, Buttons, Akzente, Links, Footer
- Wirkung: Wiedererkennbarkeit, Vertrauen, Qualität

Beispielwert:

```css
--color-primary: #005B99;
```

### Sekundärfarben

**Cremeweiß**

```css
--color-cream: #FAF6EF;
```

**Naturgrün**

```css
--color-green: #6F8F4E;
```

**Erdbeerrot**

```css
--color-berry: #C7443E;
```

**Warm Beige**

```css
--color-sand: #D9C7A3;
```

**Dunkler Text**

```css
--color-text: #1F2A2E;
```

## Typografie

### Headlines

Elegante Serifenschrift oder hochwertige Display-Schrift.

Empfehlungen:

- Playfair Display
- Cormorant Garamond
- Libre Baskerville

### Fließtext

Moderne, klare Sans-Serif-Schrift.

Empfehlungen:

- Inter
- Source Sans 3
- Lato
- Manrope

## Layout-Prinzipien

- Mobile First
- viel Weißraum
- große emotionale Bilder
- klare Call-to-Actions
- Karten für Angebote und Saisonbereiche
- sticky mobile Kontakt-/Reservierungsbutton
- schnelle Orientierung auf jeder Seite

## UI-Komponenten

### Header

Bestandteile:

- Logo links
- Navigation mittig oder rechts
- CTA „Tisch reservieren“
- Mobile Burger-Menü

### Hero Section

Bestandteile:

- großes Bild oder Video
- kurze Headline
- emotionaler Subtext
- 2 bis 3 CTA-Buttons
- aktuelle Saisoninfo als Overlay oder darunter

### Aktuell-Karte

Beispiel:

```text
Jetzt aktuell
Frische Erdbeeren, Blaubeeren & Kirschen im Hofmarkt.
Heute geöffnet: 8–18 Uhr
[Mehr erfahren]
```

### Erlebnis-Kachel

Vier Hauptkacheln:

- Hofküche
- Hofmarkt
- Hofplantagen
- Feiern

### CTA-Banner

Beispiel:

```text
Planen Sie Ihren Besuch auf dem Schmücker Hof.
Öffnungszeiten, Anfahrt und aktuelle Angebote auf einen Blick.
[Besuch planen]
```

### Footer

Footer-Spalten:

- Hofbereiche
- Saison & Specials
- Kontakt
- Rechtliches
- Social Media

## Bildsprache

Die Website braucht echte, hochwertige Bilder vom Hof.

Benötigte Motive:

- Hofeingang
- Innenhof
- Kaminsaal
- Frühstücksbuffet
- Kaffee und Kuchen
- saisonale Gerichte
- Hofmarkt
- Geschenkkörbe
- Erdbeerfelder
- Blaubeeren
- Spargel
- Selbstpflücken
- Team
- Hochzeitssituation
- Adventsstimmung

## Animationen

Dezent und hochwertig:

- sanfte Fade-ins beim Scrollen
- leichte Parallax-Wirkung im Hero
- Hover-Effekte auf Karten
- Sticky CTA auf Mobile
- animierter Saisonkalender

Keine übertriebenen Effekte. Der Hof soll modern wirken, aber nicht künstlich.
