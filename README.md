# Schmücker Hof – Pitch-Prototyp

Statischer Next.js-Prototyp für die Neugestaltung des Schmücker Hofs.

Die Website ist für einen statischen Export vorbereitet und kann über GitHub Pages veröffentlicht werden.

## Lokal starten

```bash
npm install
npm run dev
```

Danach ist die Website unter `http://localhost:3000` verfügbar.

## Umsetzungsstand

- [x] Next.js- und TypeScript-Grundlage
- [x] Responsives Designsystem und Startseite
- [x] Hero, Saison-Hinweis, Erlebnisbereiche, Reservierungs- und Besuchs-CTA
- [x] Unterseiten: Hofküche und Feiern & Hochzeiten
- [ ] Unterseiten: Hofmarkt, Hofplantagen, Saison, Jobs und Kontakt
- [ ] Reale Inhalte, Rechtliches, Formulare und CMS-Anbindung

Die Adresse, Telefonnummer, E-Mail-Adresse, Öffnungszeiten und Saisonhinweise sind im Pitch-Prototyp bewusst als Beispielinhalte angelegt und müssen vor einem Livegang durch die finalen Angaben ersetzt werden.
